Welcome To My Web Development Portfolio

A little more about me...

It started out as a passion for photography, oddly enough. I have been searching for a way to showcase my photography work to my followers on line, but the online website templates didn’t fit my style. I wanted something personal, something that was more.. Me. It dawned on me that I could just make my own. From scratch. Shouldn’t be too hard, right?


I couldn’t have been more wrong.


I shadowed with my brother in law who is an experienced Web Developer and he showed me basic code to get me started. Eventually, I was starting to invest more and more time into it and I was getting lost in the limitless opportunities to learn more. A year later, I was finally ready to build my first ever website, my Photography Page.


And that’s where it all began. From that point on, I knew Web Development is what I wanted to do.

HTML

What is HTML?


Hypertext Markup Language is the standard markup language for creating webpages.


What can you do with HTML?


  • HTML provides the structure and content of page
  • Supports many media elements

CSS

What is CSS?


Cascade Styling Sheets is a stylesheet language used to create the styling of a document written in a markup language.


What can you do with CSS?


  • Alters the presentation of web pages (color, layout, fonts, etc.)
  • Allows to adapt to different types of devices size

Javascript

What is Javascript?


Javascript is an interpreted programming language that is used to enhance HTML pages.


What can you do with JS?


  • Controls behavior of elements on the page
  • Provides dynamic and interactive experience for users

Git

What is Git?


Git is a version-control system for tracking changes in files and coordinating work on those files with multiple users.


What can you do with Git?


  • Helps coordinate teams to work on same/different files
  • Helpful in preventing merge conflicts
